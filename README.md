# AneriAI - Professional Assistant Chatbot

A RAG (Retrieval-Augmented Generation) based chatbot that answers questions about Aneri Bhavsar's professional background using her resume as the knowledge base.

## Features

- 🤖 AI-powered chat interface using Gemini 2.0 Flash
- 📄 Resume-based knowledge retrieval
- 🔍 Context-aware responses with source attribution
- 💬 Professional, first-person responses
- 📱 Mobile-responsive design
- 🔒 CORS-enabled for portfolio integration

## Tech Stack

- **Backend**: Flask, Python
- **AI/ML**: LangChain, Groq API (Llama 3.8B)
- **Vector Database**: ChromaDB
- **Embeddings**: Sentence Transformers (all-MiniLM-L6-v2)
- **Frontend**: HTML, CSS, JavaScript
- **PDF Processing**: PyPDF2
